### Main Capabilities

- Modern and responsive user interface
- Category-based navigation (Live TV, Movies, Series, Favorites)
- Smart pagination for large playlists
- Real-time search filtering
- HLS adaptive streaming support
- Multi-user profile management
- Local data persistence

ApacheTV runs entirely in the browser without requiring a backend server.
